# Amendment: Redesign Header to Match Reference Design

## Problem

Current header is minimal and unstyled. The target design shows:
- Full-width header bar (background color handled by existing light/dark mode styles)
- Left: Site title (bold) with navigation links below (underlined)
- Right: Logo image
- Professional, modern appearance

## Design Decisions

1. **Logo text**: Skip - just show the logo image, no text beside it
2. **Navigation**: Keep current 3 items (Home, About, Blog) - only style what exists
3. **Background**: Use existing light/dark mode styles (no explicit background-color)

## Implementation Plan

### Task 1: Update `_header.html` template structure
**Skill**: `go-writer`

Restructure HTML for new layout (logo only, no text beside it):
```html
{{define "_header.html"}}
<header>
    <div class="header-left">
        <a href="/" class="site-title">{{.Site.Title}}</a>
        <nav>
            {{range .Site.Navigation}}
            <a href="{{.URL}}">{{.Title}}</a>
            {{end}}
        </nav>
    </div>
    {{if .Site.Logo}}
    <div class="header-right">
        <img src="{{.Site.Logo}}" alt="{{.Site.Title}} logo" class="logo">
    </div>
    {{end}}
</header>
{{end}}
```

**File**: `internal/renderer/templates/_header.html`

### Task 2: Add header CSS styles
**Skill**: `go-writer`

Add header-specific styles to `example/assets/style.css`:

```css
/* Header styling */

```

**File**: `example/assets/style.css`
