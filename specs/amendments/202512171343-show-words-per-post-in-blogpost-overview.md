# 202512171343 - Technical amendment specification

## Objective

Show the word count on the blog listing page `/blog/`, per blog entry. 

The word count should be displayed on the same location as on an individual blog page. I.e. all the way to the right. 

The wordt count should have the same styling as on an individual blog page.

## Examples

Example /blog/ entry
```text
2025-12-08 blog-post-title                                  12 words
```

Note the word count postition is determined by the right side of the <main> element. `2025-12-08` is outlines to the left of the <main>. The word count needs to be outlined to the right side.
